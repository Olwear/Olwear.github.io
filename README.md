### 元素精通大师 —— 砂糖 

* **过气的砂糖不要扔，裹上鸡蛋液，粘上面包糠，炸至金黄，隔壁小孩都馋哭惹**

* **砂糖作为一名元素精通辅助，为队友提供高额元素精通，实现高额伤害的元素增幅反应。砂糖其附带的聚怪被相当多的玩家诟病为温迪的下位，而真正明白其意义的人不在少数**

* **武器**

    * 祭礼残章

* **圣遗物**

    * 风套4

    * 教官4

    * 主元素精通，元素充能效率次之

* **养成思路**

    * 主要提升角色等级，提高生存能力，天赋升级取决个人。无需担心资源的投入过多，也不必投入过多资源

### 王牌火箭队 —— 刻晴 | 菲谢尔 | 香菱 | 班尼特

* **针对深境大型精英怪的无敌爆破，可以通过各种无敌帧在极其安全的环境下输出，尤其对于遗迹系列的精英怪物有强大的压制作用，不仅限于遗迹精英怪**
    
* **刻晴**

    * 作为尴尬的战场C，这一次作为副C可以毫无顾及的输出，EQ完事

* **菲谢尔**

    * 奥兹发射器，六命质变（**作为未来围绕奥兹机关枪的队伍有望carry**）

* **香菱**

    * 脱手输出爆炸，安逸后台

* **班尼特**

    * 加攻王牌，奶量爆炸

* **配置要求**

    * 刻晴，菲谢尔无脑输出向，香菱适当考虑充能，班尼特主充能副生命。由于班尼特的攻击加成几乎全覆盖，全队攻击力作为次要因素

* **输出手法**

    * 班尼特EQ，香菱QE，菲谢尔E|Q,刻晴站场Q接高空E坠击，适时拉开距离，实现高度循环
